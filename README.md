# Breast Cancer Survival Prediction using Logistic Regression

This project is a machine learning classification model that predicts whether a breast cancer patient is **Alive or Dead** based on medical and clinical attributes.

The goal of this project is to apply **Logistic Regression** to a real-world healthcare dataset and strengthen understanding of classification models, data preprocessing, and model evaluation.

## Features Used

The model learns patterns from several important clinical factors including:

- Age
- Tumor Size
- T Stage
- N Stage
- 6th Stage
- Estrogen Status
- Progesterone Status

## Target Variable

The model predicts the **patient survival status**:

- `0 → Alive`
- `1 → Dead`

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

## Machine Learning Workflow

1. Data loading and exploration  
2. Target variable encoding (Alive / Dead → 0 / 1)  
3. Feature selection  
4. Categorical data encoding  
5. Train-test split  
6. Logistic Regression model training  
7. Model evaluation using accuracy

## Purpose of the Project

This project demonstrates how classification algorithms can be applied to medical datasets to identify patterns and predict survival outcomes.

It also strengthens understanding of:

- Binary Classification
- Logistic Regression
- Data Preprocessing
- Model Evaluation
